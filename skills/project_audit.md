# Skill: ProjectAudit
Description: Conducts a full audit of a project's structure and consistency.
Tools Required: list_dir, read_file
Logic Flow:
1. Map the project directory structure using `list_dir`.
2. Read `README.md` and `CLAUDE.md` to understand project goals.
3. Check if the actual file structure matches the described architecture.
4. Report any discrepancies or missing documentation.
